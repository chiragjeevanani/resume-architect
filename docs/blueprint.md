# **App Name**: Resume Architect

## Core Features:

- Structured Input Forms: Provide intuitive, structured forms for users to input their resume information, covering all standard sections like education, experience, skills, and projects.
- Real-time Preview: Generate a dynamic, live preview of the resume, mirroring the chosen template. Changes in the input forms should instantly reflect in the preview.
- Template Selection: Offer a curated selection of elegant resume templates to choose from, with a focus on clear, professional designs.
- PDF Download: Enable users to download their resume as a PDF file, ensuring the output matches the live preview's formatting and layout exactly to that from the provided sample image.

## Style Guidelines:

- Primary color: Deep blue (#1A237E), evoking professionalism and trust.
- Background color: Clean white (#FAFAFA) to provide excellent readability and a modern feel.
- Accent color: Light blue (#3F51B5), used sparingly for interactive elements.
- Headline font: 'Playfair', a modern serif giving a high-end, fashionable feel. Paired with 'PT Sans' for the body.
- Employ a clean, single-column layout with generous white space for enhanced readability.
- Use simple, professional icons to visually represent different sections (e.g., a graduation cap for education, briefcase for experience).
